Type Parameters
Name Description
T
Remarks
Requesting more than one element delegates to Shuffle<T>(IEnumerable<T>, Random), which buffers the collection into memory.

Note that you need to be careful about the instance of Random you use as:
  • Creating two instances of Random at roughly the same time will yield the same sequence of random numbers (when used in the same way)
  • Random isn't thread-safe.
See C# in Depth Chapter 12: Random for details.
